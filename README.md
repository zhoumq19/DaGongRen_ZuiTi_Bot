## 项目简介：使用AI+海量趣味文案作为知识库，使用浪潮信息的源2.0大模型，使用 RAG技术，实现了帮助打工人输出”疯感“文案的功能。

## 项目愿景：帮助用户生产超出期望的文案

## 项目特性
- 懂得打工人平时如何发疯
- 懂得趣味微信状态、朋友圈
